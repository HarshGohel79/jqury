4.CSS

  1. css() : Using css() method we can set or return one or more style properties for the selected elements.
    Syntax: $(element).css("propertyname","value"); to apply single css property
    Syntax: $(element).css({"propertyname": "value", "propertyname": "value", "propertyname": "value"}); to apply multiple style/css to element.

  2. addClass() : Using addClass() method we can add one or more classes to the selected elements.
    Syntax: $(element).addClass("class1"); to apply single css property
    Syntax: $(element).addClass("class1 class2 classs3"); to apply multiple style/css to element.

  3. removeClass() : Using removeClass() method we can remove one or more classes from the selected elements.
    Syntax: $(element).removeClass("class1"); to apply single css property
    Syntax: $(element).removeClass("class1 class2 classs3"); to apply multiple style/css to element.

  4. toggleClass() : Using toggleClass() method we can toggle between adding/removing classes from the selected elements.
    Syntax: $(element).toggleClass("class1");

3. How to create slider with animation?

thare are a three mathod to apply a animation

1. jQuery slideDown....

The jQuery slideDown() method is used to slide down an element.

syntax:
$(selector).slideDown(speed,callback);

example : br $("#flip").click(function(){ $("#panel").slideDown(); });

2. jQuery slideUp......

thare are a three mathod to apply a animation

1. jQuery slideDown....

The jQuery slideUp() method is used to slide up an element.

syntax:
$(selector).slideUp(speed,callback);

example : br $("#flip").click(function(){ $("#panel").slideUp(); });

3. jQuery slideToggle

The jQuery slideToggle() method toggles between the slideDown() and slideUp() methods.
If the elements have been slid down, slideToggle() will slide them up.
If the elements have been slid up, slideToggle() will slide them down.

syntax:
$(selector).slideUp(speed,callback);
